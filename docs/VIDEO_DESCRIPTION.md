## Full Description

This video demonstrates a complete MLOps pipeline running locally with Docker Compose. The system serves a machine learning model through FastAPI, logs experiments and model artifacts with MLflow, exposes production metrics through Prometheus, visualizes model health and concept drift in Grafana, and includes automated CI/CD workflows with GitHub Actions.

The research focus is drift-aware adaptive retraining for streaming classification. The project includes a HybridDD drift detector, benchmark experiments, statistical analysis, latency and throughput evaluation, reproducibility notes, monitoring dashboards, architecture diagrams, screenshots, and an IEEE-format research paper.

What is shown in the live demo:

1. FastAPI Swagger documentation at `localhost:8000/docs`.
2. API health endpoint.
3. Live `POST /predict` request and response.
4. Prometheus scrape targets and prediction metrics.
5. MLflow experiment tracking evidence.
6. Grafana model performance dashboard.
7. Grafana drift and adaptive retraining dashboard.
8. Grafana infrastructure and API health dashboard.
